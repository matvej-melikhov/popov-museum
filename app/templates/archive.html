{{ _('Хронология') }}

1859
{% if get_locale() == 'en' %}

Born 16 March in the settlement of Turyinskie Rudniki, Perm Province, into the family of priest Stepan Petrovich Popov.

{% else %}

Родился 16 марта в посёлке Турьинские Рудники Пермской губернии в семье священника Степана Петровича Попова.

{% endif %}
1877
{% if get_locale() == 'en' %}

Graduated with distinction from the Perm Theological Seminary. Enrolled in the physics and mathematics faculty of St. Petersburg University.

{% else %}

Окончил с отличием Пермскую духовную семинарию. Поступил на физико-математический факультет Петербургского университета.

{% endif %}
1882
{% if get_locale() == 'en' %}

Graduated from the university and defended his dissertation. Accepted a teaching post at the Mine Officer Class in Kronstadt — the beginning of 18 years of naval service.

{% else %}

Окончил университет, защитил диссертацию. Принял должность преподавателя в Минном офицерском классе в Кронштадте — начало 18-летней службы во флоте.

{% endif %}
1895
{% if get_locale() == 'en' %}

7 May — at a meeting of the Physics Section of the RPCS in St. Petersburg the radio receiver was demonstrated for the first time. This date became Radio Day.

{% else %}

7 мая — на заседании Физического отделения РФХО в Петербурге впервые продемонстрирован радиоприёмник. Эта дата стала Днём радио.

{% endif %}
1896
{% if get_locale() == 'en' %}

24 March — between the buildings of St. Petersburg University, over a distance of 250 metres, the world's first radiogram was transmitted: "Heinrich Hertz".

{% else %}

24 марта — между корпусами Петербургского университета на 250 метров передана первая в мире радиограмма: «Генрих Герц».

{% endif %}
1900
{% if get_locale() == 'en' %}

The first practical radio link Gogland — Kotka was opened. The very first radiogram saved 27 fishermen on an ice floe. At the World Exhibition in Paris — gold medal.

{% else %}

Введена первая практическая линия радиосвязи Гогланд — Котка. Первая же радиограмма спасла 27 рыбаков на льдине. На Всемирной выставке в Париже — золотая медаль.

{% endif %}
1901
{% if get_locale() == 'en' %}

Appointed full professor of physics at the Electrotechnical Institute (ETI). Founded the physics laboratory and established 42 laboratory practicals.

{% else %}

Принял должность ординарного профессора физики в Электротехническом институте (ЭТИ). Основал физическую лабораторию и поставил 42 лабораторные работы.

{% endif %}
1903
{% if get_locale() == 'en' %}

Attended the Berlin International Radio Conference as Russia's representative. German Postmaster General: "To Popov we owe the emergence of the first radiographic apparatus."

{% else %}

Участвовал в Берлинской международной конференции по радиосвязи как представитель России. Министр почт Германии: «Попову мы обязаны возникновением первого радиографического аппарата».

{% endif %}
1905
{% if get_locale() == 'en' %}

The ETI Council elected Popov director of the institute following the introduction of university autonomy. He signed a protocol supporting democratic freedoms for students.

{% else %}

Совет ЭТИ избрал Попова директором института после введения университетской автономии. Подписал протокол в поддержку демократических свобод студентов.

{% endif %}
1906
{% if get_locale() == 'en' %}

Died 13 January in St. Petersburg from a cerebral haemorrhage. Buried at the Literatorskie Mostki section of the Volkovo Cemetery.

{% else %}

Скончался 13 января в Санкт-Петербурге от кровоизлияния в мозг. Похоронен на Литераторских мостках Волковского кладбища.

{% endif %}
1945
{% if get_locale() == 'en' %}

By decree of the Soviet Government, 7 May was declared an annual holiday — Radio Day. The A.S. Popov Gold Medal of the USSR Academy of Sciences was established.

{% else %}

Постановлением Правительства СССР 7 мая объявлен ежегодным праздником — Днём радио. Учреждена Золотая медаль имени А.С. Попова Академии наук СССР.

{% endif %}
